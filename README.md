# TTM

TTM is a simple tiling task manager that uses vim/calcurse key bindings.

### Key Bindings

- n -> new column
- a -> add item
- A -> add scheduled item
- d -> remove item
- D -> remove column
- j -> select below item
- k -> select above item
- l -> select next column
- h -> select previous column
- J -> swap below item
- K -> swap above item
